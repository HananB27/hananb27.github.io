Recent Projects

Project 1

Google Search

For this project, I created a website with three pages: a Google Search page, an Image Search page, and an Advanced Search page. The Search page featured a search bar and button that directed the user to Google's search results for their query. The Image and Advanced Search pages also had search functionality, with the latter featuring four text fields for advanced search options. The main Search page had an "I'm Feeling Lucky" button that took the user directly to the first search result for their query. The website's design was consistent with Google's aesthetics and the CSS closely resembled Google's own.

Project 2

Auction site

I implemented an auction site with the following features: creating new listings, viewing active listings, viewing a specific listing's details, adding items to a watchlist, bidding on items, closing auctions, viewing closed auctions, adding comments to listings, viewing listing categories, and managing the site via the Django admin interface. Users can also view their own watchlist. The site includes models for listings, bids, and comments.

Project 3

Wiki site

I developed a comprehensive Wiki encyclopedia with various capabilities including accessing and searching entries, creating and editing entries, accessing random entries, and converting Markdown to HTML. The Entry and Index Pages enable access to specific entries, while the Search function allows users to search for and access matching entries. Users can also create new entries and edit existing ones using the New Page and Edit Page features. The Random Page feature takes users to a random entry, and the Markdown to HTML conversion feature converts Markdown content to HTML for display.
